今天来看下Suno Api的视频封面编辑。

Suno AI会被生成的音乐顺带生成一个带有字幕的MV文件，格式为MP4。

对于此mv图片是默认生成的，可能想讲视频封面换成帅帅的自己的照片，那么也是支持的。
